  • Enable Sync Requests Menu: Show/Hide Sync Requests Menu in Ektefa Logs Menu.

How It Works

Easily access Ektefa Payroll features from your Odoo dashboard

Configurations

Configure your Ektefa credentials in Odoo settings and these fields are required.

  • API Key: Unique API key for accessing Ektefa API.
  • API Secret: Secret key for accessing Ektefa API.
  • Odoo Company: Select the Odoo company with Ektefa Company ID.
  • Odoo Accounting Settings:
    • Auto Posted Journal Entries: Enable/Disable auto posted journal entries.
    • Default Journal: Select the default journal for Ektefa.

Automations

Start syncing your payroll data automatically
